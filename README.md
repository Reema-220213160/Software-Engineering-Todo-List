# To-Do List Application - مساق هندسة البرمجيات

مشروع جماعي يهدف لتطبيق مفاهيم إدارة المشاريع البرمجية باستخدام Git .


 
## 👥 فريق العمل
المشروع من إنجاز المجموعة المكونة من 9 طالبات، حيث تم توزيع المهام بناءً على مبدأ Feature-based Division 

| الطالبة | الميزة (Feature Branch) | المسؤولية البرمجية |
| :--- | :--- | :--- |
| الطالبة 1 | feature/structure-html | هيكلة ملف index.html وتوزيع الـ div الأساسي. |
| الطالبة 2 | feature/css-layout | التنسيق العام في style.css (الـ container والـ title). |
| الطالبة 3 | feature/css-input-btn | تنسيق الـ input والـ button في ملف style.css. |
| الطالبة 4 | feature/css-list-items | تنسيق الـ ul والـ li والـ span (الحذف). |
| الطالبة 5 | feature/css-checked-state | إدارة الصور (checked.png) وحالة الـ .checked في الـ CSS. |
| الطالبة 6 | feature/js-add-task | برمجة دالة addTask داخل ملف script.js. |
| الطالبة 7 | feature/js-delete-task | برمجة منطق الحذف في event listener داخل script.js. |
| الطالبة 8 | feature/js-localstorage | إدارة دالتي setTask و loadTasks في script.js. |
| الطالبة 9 | feature/js-toggle-checked | إدارة تبديل الحالة classList.toggle داخل script.js. |
